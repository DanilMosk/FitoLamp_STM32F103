# FitoLamp_STM32F103
Проект домашней фитолампы с автоматическим выключение по заданному времени на основе микроконтроллера STM32F103

В проекте реализовано установка текущего времени при подаче питания, настройка времени включения и выключения лампы. Время демонстрируется на 4-х циферном семисегментном индикаторе. В проекте используются 4 кнопки: 1 - выбор настраиваемого времени, 2 и 3 - кнопки увелчиения/уменьшения времени, 4 - переключение между часами/минутами. 

В проекте используются пины:
PB7 - D1
PB3 - D2
PB4 - D3
PB5 - D4
PA0 - кнопка 1
PA2 - кнопка 2
PA3 - кнопка 3
PA6 - кнопка 4
PA4 - Lamp1
PB12 - сегсент A
PB13 - сегсент B
PB14 - сегсент C
PB15 - сегсент D
PA8 - сегсент E
PA9 - сегсент F
PA11 - сегсент DP
