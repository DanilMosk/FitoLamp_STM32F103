# FitoLamp_STM32F103
Проект домашней фитолампы с автоматическим выключение по заданному времени на основе микроконтроллера STM32F103

В проекте реализовано установка текущего времени при подаче питания, настройка времени включения и выключения лампы. Время демонстрируется на 4 циферном семисегментном индикаторе. В проекте используются 4 кнопки: 1 - выбор настраиваемого времени, 2 и 3 - кнопки увелчиения/уменьшения времениб 4 - переключение между часами/минутами.
